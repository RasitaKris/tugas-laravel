## Fitur Keranjang Belanja dan Checkout

Project ini dibuat untuk memenuhi tugas **Assignment AFL 3** pada mata kuliah Web Development menggunakan framework **Laravel**.

### Deskripsi Singkat
Aplikasi web ini mengimplementasikan fitur **keranjang belanja (shopping cart)** dan **checkout** yang terintegrasi dengan database. User diwajibkan login untuk melakukan transaksi pembelian dan dapat melihat riwayat pembelian yang telah dilakukan.

### Fitur yang Diimplementasikan
1. Data keranjang belanja dan pembelian disimpan dalam database  
2. User harus login untuk mengakses keranjang belanja  
3. User dapat menambahkan produk ke keranjang belanja  
4. User dapat mengubah jumlah produk di keranjang belanja  
5. User dapat menghapus produk dari keranjang belanja  
6. Tersedia tampilan subtotal dan total pembelian  
7. User dapat melakukan checkout dengan mengisi alamat pengiriman dan metode pembayaran  
8. User dapat melihat daftar pembelian (order history)  

### Catatan
- Seluruh fitur telah di-*push* ke branch **main**
- Aplikasi dapat dijalankan tanpa error
